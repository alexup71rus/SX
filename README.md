## Концепт игры

Это игра в жанре jRPG.
Игрок управляет отрядом.
Перемещение по миру происходит в режиме миникарты.
Диалоги происходят в режиме визуальной новеллы.
Бои происходят в режиме пошаговой стратегии (jRPG).
